Stretch Questions

1. What is the difference between an inline element and a block element?
-Inline element can be used inline with a set of text and doesn't create
a new section of code like a block element.  For instance:
<p>Hello my name is <span>Blake</span></p> would be an inline element:
Hello my name is Blake
A block element would break up the structure:
<div>Hello my name is <p>Blake</p></div>
Hello my name is 

Blake
